# Information Gathering

## Dmitry  
Find domain public ip  
``
$ dmitry -w domain  
``  

Find information about domain  
``
$ dmitry -i domain  
``

Find domain-associated emails  
``
$ dmitry -e domain  
``

Find subdomains  
``
$ dmitry -w domain  
``
